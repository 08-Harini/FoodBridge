--------------------------------------------------

Overview

FoodBridge is a web-based platform developed to connect individuals and organizations with surplus food to those who need it. The platform aims to reduce food wastage by providing a simple and efficient system for donating and accessing food.

It enables users to share available food resources while ensuring ease of use, accessibility, and seamless interaction between donors and receivers.

--------------------------------------------------

Features

User Roles

Donor
- Add food donations with detailed information
- Manage and track listed donations
- View and respond to incoming requests

Receiver
- Browse available food listings
- View detailed information about each listing
- Request food based on specific requirements
- Track the status of submitted requests

--------------------------------------------------

Core Functionalities

- Browse food listings with search and filtering capabilities
- Add food donations with details such as type, quantity, location, and expiry
- Access detailed information for each food listing
- Submit requests with quantity and purpose
- Dashboard to monitor activities and interactions
- Notification system for updates and request status
- Option to cancel pending requests
- Expiry tracking to ensure food availability accuracy
